Artificial Intelligence - Javascript


This project is coded entirely in JavaScript, implemented through an HTML page. It is a graphical representation of a car using artificial intelligence to determine if a light is green or not, and using that decision to determine whether or not to cross a busy intersection. The smart car uses the Hidden Markov Model (HMM) to determine the state of the light at the intersection. The cross traffic is not using an AI algorithm, and is hard coded to simply go when they have a green light. This project helped solidify my JavaScript skills, as well as serving as my first implementation of artificial intelligence in my code.

Conway's Game of Life


This program, written in Java, is a classic example of Conway's Game of Life. It is essentially a program which utilizes Javafx to create a game board initialized with a random number of cells alive. The game then goes through 100 iterations following the rules of Conway's game of life, which can be found here.
This project was my first time programming with javafx, and was a good way to cement my understanding of object oriented programming, since I was working mainly with classes and objects that I was unfamiliar with.
